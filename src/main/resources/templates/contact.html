Oehotin

This web-application was first made for Helsinki University web-server programming course. After that I have incorporated new things I have learned to it.

This is build with Netbeans and its backend is mostly Java. Other used technologies: Spring, Thymeleaf, Maven, JS, CSS, HTML and Bootstrap.

Oehotin follows Layered Architecture: UI calls controller layer, controller layer calls service layer which calls repositories. Repositories handle database. No layer is skipped.

Things to improve:

  • Change picture storage to some external place (like AWS S3) and only save link to that in database
